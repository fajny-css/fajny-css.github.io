Fanjy CSS uses its own icons. Most of icons have two weights: regular and solid. When using them, try to stick to one weight. We're using here an icon sprite, generated with Icomoon.

How to use

In HTML

Copy the code and insert it in your HTML page.

{% include demo-code.html language="html" id="demo-icon-html" content='<svg class="icon"> <use xlink:href="/icons/icon-sprite.svg#icon-name"></use> </svg>' %}

In SCSS

You can use mixins to include your icon. Here, the color variable is not required.

{% include demo-code.html language="scss" id="demo-icon-scss" content='div:after { @include icon(icon-name, size, color) }' %}
