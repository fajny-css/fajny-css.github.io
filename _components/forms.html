Forms have different components. Since Fajny does not use JavaScript some components can not be properly designed. An error text is already coded, but you would have to use a JS form checker.

File to include

@include "components/forms";

Inputs

Email is required
Email must be a valid email address
<div class="input-container">
     <input type="text" id="input1" name="input" required />
     <span class="bar"></span>
     <label for="input1">Email</label>
     <div class="error">
          <div>Email is required</div>
          <div>Email must be a valid email address</div>
     </div>
</div>
