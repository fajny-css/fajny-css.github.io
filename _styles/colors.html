--- layout: detail title: Colors desc: Colors for Fajny CSS ---

Fajny CSS has its own color palette. If you want to use your own colors, you can edit them in the file scss/_colors.scss.
You can find a nice color palette generator here: https://coolors.co/.

@import "colors";
