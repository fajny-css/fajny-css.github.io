--- layout: default title: Icons desc: Icon font for Fajny CSS ---

Fanjy CSS uses its own icon font. If you want to use it add this in the head of your HTML:
<link href="css/fajny-icons.min.css" rel="stylesheet" />
or via CDN
<link rel="stylesheet" href="https://unpkg.com/fajny-css-icons@1.0.0/css/fajny-icons.min.css" />

How to use

In the HTML

Copy the code and insert it in your HTML page.

Example
<i class="icon icon-user-regular"></i>

In the SCSS

You can use mixins to include your icon.

Example
@include icon("\unicode")

In the CSS

Copy this in your CSS page:

font-family: "fajny-css-icons";
content: "\unicode";
