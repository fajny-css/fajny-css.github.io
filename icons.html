Fanjy CSS uses its own icon font. Most of icons have two weights: regular and solid. When using them, try to stick to one weight.

If you want to use this icon font add this in the head of your HTML: 

{{ site.icons_local }}

or via CDN

{{ site.icons_cdn }}

How to use

In the HTML

Copy the code and insert it in your HTML page.

Example
<i class="icon icon-user-regular"></i>

In the SCSS

You can use mixins to include your icon.

Example
@include icon("\unicode")

In the CSS

Copy this in your CSS page:

font-family: "fajny-css-icons";
content: "\unicode";
